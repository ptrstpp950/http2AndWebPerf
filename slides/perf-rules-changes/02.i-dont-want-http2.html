I don't want HTTP/2

HTTP/2 is backwards-compatible with HTTP/1.1, but it is a bit faster.

Response times limits (Usability Engineering - Jakob Nielsen, 1993)

  • 0-100 ms - Instant
  • 100-300 ms - Sluggish
  • 300 - 1000 ms - Is working
  • 1s+ - Mental context switch
  • 10s+ - I'll come back later...
